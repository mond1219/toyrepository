# 자료구조 

`데이터를 저장하고 관리하는 방식`

데이터를 `체계적`으로 저장하여 `메모리`를 `효율`적으로 사용, 빠르고 `안정적`인 `데이터 처리`에 도움

- 선형 자료구조 : array, dynamic array, linked list, queue, stack, hash table
- 비선형 자료구조 : tree, graph
